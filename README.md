## Companion tool: Application Autofill

Once you've found and applied to a job through the engines above, [Simplify_Job](https://github.com/kunal202426/Simplify_Job)
is a separate Chrome extension that fills out the actual application form for you on the ATS
portal (Greenhouse, Lever, Workday, and most others) — and learns from every question you
answer manually, so reworded versions of the same question get filled automatically next
time. It's local-first: your profile and everything it learns stay on your machine, no
account or network calls required. Each user configures it with their own details; nothing
is shared between installs. Install it separately and set up your own profile in its popup —
this repo doesn't bundle or run it for you.
